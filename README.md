# Sleepy-Course-Selector

I need sleep, so a scrapy helps me select course.



## How-to use

```shell
python3 xk.py --xh MG20330064 --pw 34D2C0A9BDCE3EAD13E80D55CAE0C457574C9DECDA63AD2C --st 0.1
```



> xh: 学号
>
> pw: 加密后的48位密码，可以F12查看
>
> st: 抢课间隙，做人留一地
